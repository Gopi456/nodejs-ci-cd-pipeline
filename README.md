# Node.js CI/CD Pipeline using GitHub Actions & Docker

## Tech Stack:
- Node.js
- Express.js
- Docker
- GitHub Actions

## Workflow:
1. Triggered on push to `main` branch
2. Runs build, test, Docker build & Docker push
3. Deploys image to DockerHub
